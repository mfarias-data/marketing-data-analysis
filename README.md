Marketing Campaign & Lead Tracking System (SQL + Power BI)
📋 Overview
This project demonstrates a data-driven approach to managing marketing leads and evaluating campaign performance. It bridges the gap between technical database management and strategic marketing insights by simulating a real-world business scenario where lead conversion and ROI need to be tracked accurately.

🎯 Business Objective
The goal is to provide marketing managers with a clear view of which channels (Email, Social Media, Paid Ads) are delivering the highest quality leads and the best return on investment (ROI), allowing for data-backed budget allocation.

🛠 Tech Stack
Database: MSSQL Server (Data Modeling, Schema Creation, and Complex Queries).

Data Visualization: Power BI (Interactive Dashboards & DAX).

Concepts: ETL (Extract, Transform, Load), Relational Mapping, and Marketing KPIs.

🚀 Key Features
Relational Database Schema: * Leads table: Captures customer information and source.

Campaigns table: Tracks budget, duration, and channel.

Conversions table: Links successful sales to specific marketing efforts.

SQL Analysis: Scripts to calculate Conversion Rates, Cost per Lead (CPL), and Customer Acquisition Cost (CAC).

Power BI Dashboard: A visual representation of the sales funnel, showing lead progression from initial contact to final sale.

📂 Repository Structure
/scripts: SQL files for database creation and analysis.

/data: Sample datasets (CSV/Excel) used to populate the database.

/dashboard: Power BI file (.pbix) or link to the interactive report.
